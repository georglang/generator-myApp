Usage
======
- clone repo
- npm link
- execute "yo myApp" to create a new application
- run grunt server 


Installation options:
=====================
- Bootstrap
- jQuery
- AngularJs


Grunt Tasks:
============
grunt server
grunt test (jasmine)
grunt hint (jshint)
grunt minifyJs (uglify)
grunt minifyCss (cssmin)


Included:
=========
- filewatcher: index.html, *.css, *.js, Spec.js, minifyJs, minifyCss
- livereload
- jasmine
- sass
- jshint
- uglify
- cssmin
